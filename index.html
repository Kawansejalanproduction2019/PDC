Arsitektur Hybrid: Desktop GUI & Flask API

PDC beroperasi dalam mode ganda, memberikan fleksibilitas operasional yang maksimal.

1. Absensi Multi-Klien Fleksibel

Sistem utama (GUI Desktop) berjalan berdampingan dengan **Flask API Server** (`launcher.py`). Ini memungkinkan Absensi Wajah/Barcode dari HP/Web, sehingga kamera eksternal tidak diperlukan.

2. Stabilitas Multithreading Inti

Deteksi wajah, pemrosesan video, dan Flask API berjalan di **Thread** yang terpisah. Ini memastikan GUI tidak pernah *freeze* dan absensi *real-time* tetap responsif saat tugas berat (seperti AI) diproses.

3. Komunikasi Dua Arah

Data Absensi dicatat di **GUI/API**, dan sinyal notifikasi dikirim ke Discord Bot melalui **File Queue** (`notif_queue.json`). Arsitektur terpisah ini menjamin ketahanan sistem: jika salah satu komponen *crash*, yang lain tetap berjalan.

Arsitektur Deployment yang Kuat & Efisien

Sistem kini memiliki struktur Tiga Komponen yang tangguh: GUI, Flask API, dan Discord Bot.

Kesimpulan Arsitektur

PDC menawarkan stabilitas, multi-fungsi, dan fleksibilitas *deployment* melalui arsitektur Hybrid yang memungkinkan absensi dari PC utama maupun perangkat mobile.
